지금 네이버 스포츠(경기 일정/결과) 페이지 크롤링이 차단당하고 있어.
단순 요청 방식은 버리고, 'Puppeteer(퍼피티어)'를 사용한 [강력한 우회 모드]로 코드를 다시 짜.

[요구 사항]
1. **Target:** 네이버 스포츠 페이지 로딩이 끝날 때까지 기다릴 것 (`networkidle2`).
2. **Stealth Mode (위장술):**
   - User-Agent를 랜덤하게 변경해서 일반 PC 사용자인 척해.
   - `webdriver` 속성을 숨겨서 자동화 툴인 걸 들키지 않게 해.
3. **Retry Logic (막히면 뚫기):**
   - 만약 데이터를 못 찾거나 차단당하면(Error), 즉시 포기하지 말고 2초 대기 후 최대 3번까지 '재시도(Retry)' 하는 로직을 넣어.
4. **Data Extraction:**
   - 화면에 렌더링된 경기 시간, 팀 이름, 점수(결과) 텍스트를 정확히 추출해서 JSON으로 반환해.

지금 당장 `npm install puppeteer` 부터 시작해서, 위 로직이 완벽하게 들어간 `app/api/naver-crawl/route.ts` 코드를 작성해.
