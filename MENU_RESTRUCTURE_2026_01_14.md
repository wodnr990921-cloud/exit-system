## 🎯 주요 변경사항

### 1. **업무관리** (신규 통합 메뉴)
**통합 대상:**
- 재무관리
- 일일 마감
- 월말 정산
- 휴면 포인트
- 소모품 재고

**구현 방식:**
- 5개 탭으로 구성된 단일 페이지
- 각 탭은 기존 컴포넌트를 동적 임포트하여 재사용
- 아이콘: 💼 (Briefcase)
- 색상: emerald (재무 관련)
- 권한: operator, ceo, admin

**파일:**
- `src/app/dashboard/operations/page.tsx` (신규)
- `src/app/dashboard/operations/operations-client.tsx` (신규)

### 2. **회원관리** (확장 통합 메뉴)
**통합 대상:**
- 회원관리 (기존)
- 수용자 관리
- 블랙리스트 (이미 통합되어 있었음)

**구현 방식:**
- 3개 탭으로 구성 (회원관리 / 수용자 관리 / 블랙리스트)
- 기존 회원관리 페이지를 확장
- 아이콘: 👥 (Users)
- 색상: purple (관리 관련)

**파일:**
- `src/app/dashboard/members/page.tsx` (수정)
- `src/app/dashboard/members/members-client.tsx` (수정)

---

### 3. **문의/답변** (신규 통합 메뉴)
**통합 대상:**
- 내 작업 목록
- 원본 파기
- 반송 처리

**구현 방식:**
- 3개 탭으로 구성된 단일 페이지
- 각 탭은 기존 컴포넌트를 동적 임포트하여 재사용
- 아이콘: 💬 (MessageSquare)

**파일:**
- `src/app/dashboard/qa/page.tsx` (신규)
- `src/app/dashboard/qa/qa-client.tsx` (신규)

---

### 4. **댓글/답글 구분 기능** (신규 기능)
**목적:**
- 내부 직원 간 소통용 **댓글 (Comment)**
- 회원에게 발송되는 공식 **답글 (Reply)**

**구현 내용:**

#### 데이터베이스 스키마 변경
```sql
-- task_comments 테이블에 추가된 컬럼
- comment_type: 'internal' | 'reply'
- sent_to_member: boolean
- sent_at: timestamp
```

**마이그레이션 파일:** `schema_migration_qa_comments.sql`

#### UI 변경
- 댓글/답글 작성 시 타입 선택 버튼
- 댓글: 회색 배경, "💬 댓글 (내부)" 버튼
- 답글: 파란색 배경, "📧 답글 (회원)" 버튼
- 각 댓글에 타입 배지 표시
- 답글 발송 여부 표시 (✓ 발송됨)

**파일:**
- `src/app/dashboard/intake/intake-client.tsx` (수정)
